# MERN Blogging Platform

A full-stack blogging platform built with **MongoDB, Express, React, and Node.js**.  
Users can **register, login, create, edit, and delete blogs**.  
The project uses **JWT authentication** to protect routes.  

## Features
- User registration and login with JWT
- Dashboard displaying all blogs
- Create new blog posts
- Edit and delete existing blogs
- SPA (Single Page Application) experience with React Router

## Project Structure
- `backend/` → Node.js + Express API, MongoDB integration
- `frontend/` → React application

## Setup
1. Clone the repository
2. Run `npm install` in both `backend` and `frontend`
3. Set up `.env` for backend (MongoDB URI & JWT secret)
4. Start backend: `npm run dev`
5. Start frontend: `npm start`
